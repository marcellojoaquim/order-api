# Order-API

## Descrição do Projeto

<p> API desenvolvida em Nodejs para gerenciamento de Pedidos(Order)</p>

## Como Baixar o Repositório
```bash
git clone https://github.com/marcellojoaquim/order-api.git
cd order-api

execute o comando:
    docker-compose up -d
    npm run dev
    estara rodando na porta 3000 a api em node e na 5432 o banco de dados.
```

### Acesso a documentação da API

` http://localhost:3000/api-docs/#/ `
<p> Você pode utilizar a collection postman na raiz do projeto para executar testes. </p>

## Tecnologias
- NodeJS
- Express.js
- JWT
- BCryptjs
- Sequelize
- PostgreSQL
- Docker + Docker-compose (Opcional)
- SwaggerUI
- Postman

<p>No caso de optar por não usar o docker, ainda será necessário ter um banco de dados postgresql 
sendo executado na porta 5432</p>
